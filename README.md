# 100件事 - 情侣打卡小程序

为爱人做100件事，记录每一份甜蜜 💕

## 项目结构

```
do-things/
├── miniprogram/     # 微信小程序前端
├── backend/         # Python + MySQL 后端 API
└── admin/           # 后台管理系统
```

## 功能特性

- ✅ AI 生成100件事清单（输入：性别、关系阶段、对方兴趣）
- ✅ 任务清单展示与进度追踪
- ✅ 打卡功能（拍照 + 文字记录）
- ✅ 微信分享/转发
- ✅ 微信一键登录

## 技术栈

| 端 | 技术 |
|---|---|
| 小程序 | 微信小程序原生开发 |
| 后端 | Python + Flask + SQLAlchemy |
| 数据库 | MySQL |
| AI | MiniMax 大模型 |
| 部署 | 腾讯云 |

## 后端配置

1. 进入 backend 目录
2. 复制配置模板：
   ```bash
   cp .env.example .env
   ```
3. 编辑 `.env` 填写配置：
   - `WECHAT_APPID` - 微信小程序 AppID
   - `WECHAT_SECRET` - 微信小程序 Secret
   - `MINIMAX_API_KEY` - MiniMax API Key
   - `MYSQL_HOST` - MySQL 主机
   - `MYSQL_PASSWORD` - MySQL 密码

4. 初始化数据库：
   ```bash
   python init_db.py
   ```

5. 启动服务：
   ```bash
   python app.py
   ```

## Logo 设计方案

由于图像生成服务限制，提供以下设计方案供参考：

### 方案一：双心叠融款
- **概念**：两个轻微重叠的爱心，组成"100"的数字轮廓
- **配色**：
  - 主色：柔粉 `#FFB6C1`
  - 辅色：珊瑚粉 `#FF6B6B`
  - 点缀：薄荷绿 `#98D8C8`
  - 背景：纯白 `#FFFFFF`
- **说明**：简洁现代，用两颗心的重叠表达情侣间的交融

### 方案二：情侣剪影打卡款
- **概念**：两个小人剪影（情侣轮廓）手牵手形成心形，外圈环绕"100"
- **配色**：
  - 主色：樱花粉 `#FFAFC5`
  - 辅色：薰衣草紫 `#C9A0DC`
  - 点缀：暖阳金 `#FFD93D`
  - 背景：纯白 `#FFFFFF`
- **说明**：温馨浪漫，情侣剪影传递甜蜜感，打卡徽章元素强调"完成"

### 方案三：礼物印章款
- **概念**：圆形印章造型，中心是"100"数字，下方有✓打卡标记
- **配色**：
  - 主色：薄荷绿 `#7FDBCA`
  - 辅色：蜜桃粉 `#FFAAA5`
  - 点缀：天空蓝 `#5DADEC`
  - 背景：纯白 `#FFFFFF`
- **说明**：清新活力，印章形式有"约定"和"认证"的仪式感
